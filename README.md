# Judul Program
Kalkulator Sederhana

# Pendahuluan 
Perangkat lunak adalah kumpulan instruksi yang membuat komputer mampu menjalankan fungsi tertentu. Salah satu contoh sederhana yang sering digunakan adalah program kalkulator. Program ini berfungsi membantu pengguna melakukan perhitungan aritmatika secara cepat dan tepat, seperti penjumlahan, pengurangan, perkalian, dan pembagian. Dengan adanya kalkulator, proses perhitungan menjadi lebih praktis dan mengurangi kemungkinan kesalahan.

# Fitur Utama 
- Menampilkan daftar pilihan operasi (tambah, kurang, kali, bagi).
- Memiliki fungsi khusus untuk setiap operasi aritmatika.
- Menerima input pilihan operasi dari pengguna.
- Menerima dua input angka untuk dihitung.
- Menghasilkan output sesuai jenis operasi yang dipilih.
- Menyertakan pengecekan error untuk pembagian dengan nol.
- Memberikan pesan jika pengguna memasukkan pilihan yang tidak valid.

# Panduan Menjalankan
Jalankan program dengan menekan Run
Setelah program di jalankan layar akan menampilkan menu seprti ini:
Pilih operasi:
- Tambah (+)
- Kurang (-)
- Kali (*)
- Bagi (/)

Program akan meminta :

Masukkan angka pertama:

Ketik angka pertama, lalu tekan enter
Program meminta lagi :

Masukkan angka kedua:
Ketik angka kedua lalu Entar.

Hasil perhitungan langsung muncul

# Dokumentasi Teknis
![GAMBAR FLOWCHART](<Flowchart.png>)

Flowchart tersebut menggambarkan alur kerja dari program kalkulator sederhana. Program dimulai dengan menampilkan menu operasi yang tersedia, yaitu tambah, kurang, kali, dan bagi. Setelah itu pengguna diminta memasukkan pilihan operasi dengan mengetik angka 1 sampai 4. Jika pengguna memasukkan pilihan yang valid, program akan meminta input berupa angka pertama dan angka kedua sebagai data untuk dihitung. Berdasarkan pilihan yang diberikan, program kemudian menjalankan fungsi yang sesuai, seperti penjumlahan, pengurangan, perkalian, atau pembagian. Jika pilihan yang dimasukkan tidak sesuai dengan daftar menu, program langsung menampilkan pesan bahwa pilihan tidak valid. Setelah operasi dilakukan, hasil perhitungan ditampilkan kepada pengguna. Terakhir, flowchart menunjukkan bahwa proses program akan berakhir setelah hasil ditampilkan atau setelah pesan kesalahan diberikan.
