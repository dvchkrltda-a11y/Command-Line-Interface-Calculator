# Judul Program
Kalkulator Sederhana

# Pendahuluan 
Perangkat lunak adalah kumpulan instruksi yang membuat komputer mampu menjalankan fungsi tertentu. Salah satu contoh sederhana yang sering digunakan adalah program kalkulator. Program ini berfungsi membantu pengguna melakukan perhitungan aritmatika secara cepat dan tepat, seperti penjumlahan, pengurangan, perkalian, dan pembagian. Dengan adanya kalkulator, proses perhitungan menjadi lebih praktis dan mengurangi kemungkinan kesalahan.
